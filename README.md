## Présentation

**SchoolTrack** est un système de gestion automatisée et "offline-first" des présences pour les voyages scolaires, développé dans le cadre d'un Travail de Fin d'Études en Bachelier en Technologies de l'Informatique à l'EPHEC. 

Ce projet s'articule autour d'une application Flutter (Mobile et Web) couplée à une API FastAPI, avec un fonctionnement hybride de scan par QR Code et NFC.

## Stack Technologique

* **Frontend (App Mobile Enseignants & Dashboard Web) :** Flutter (Dart)
* **Backend (API REST) :** FastAPI (Python)
* **Bases de données :** PostgreSQL (Serveur central) & SQLite (Stockage local chiffré)
* **Infrastructure :** Docker, Docker Compose, Traefik
