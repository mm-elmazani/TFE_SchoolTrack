# Animations Lottie — SchoolTrack Teacher App

Ajouter ici les fichiers JSON Lottie pour les feedbacks visuels du scan (US 2.2).

## Fichiers requis

| Fichier | Usage |
|---------|-------|
| `success.json` | Animation verte — scan réussi |
| `warning.json` | Animation orange — doublon (élève déjà présent) |
| `error.json` | Animation rouge — badge non reconnu |

## Sources suggérées

- https://lottiefiles.com/search?q=success+checkmark (gratuit)
- https://lottiefiles.com/search?q=warning
- https://lottiefiles.com/search?q=error+cross

> Sans ces fichiers, l'app affiche une icône Material à la place (fallback dans _LottieOrIcon).
