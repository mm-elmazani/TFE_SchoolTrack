# Sons — SchoolTrack Teacher App

Ajouter ici les fichiers MP3 pour les feedbacks sonores du scan (US 2.2).

## Fichiers requis

| Fichier | Usage |
|---------|-------|
| `beep_success.mp3` | Scan réussi (1er scan d'un élève) |
| `beep_warning.mp3` | Doublon (élève déjà scanné) |
| `beep_error.mp3` | Badge non reconnu / erreur |

## Sources suggérées

- https://mixkit.co/free-sound-effects/beep/ (licence gratuite)
- https://freesound.org (licence CC0)

> Sans ces fichiers, l'app fonctionne en mode silencieux (try/catch dans ScanProvider).
